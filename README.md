# 4d-plugin-metaphone

based on [slacy/double-metaphone](https://github.com/slacy/double-metaphone) with MacRoman wrapper for Ç and Ñ

```4d
metaphones:=DoubleMetaphone(word)
```

|Parameter|Type|Description|
|-|-|-|
|word|Text||
|metaphones|Collection of Text||
